## Controller quirks

Some controllers require special handling to correct button layout differences or non-standard report formats. BlueRetro applies these quirks automatically based on the device name or VID/PID.

<AccordionGroup>
  <Accordion title="8BitDo controllers">
    Several 8BitDo controllers are identified by quirk flags and receive adjusted button mappings:

    | Quirk flag | Controller |
    |------------|------------|
    | `BT_QUIRK_8BITDO_N64` | 8BitDo N64 controller |
    | `BT_QUIRK_8BITDO_N64_MK` | 8BitDo N64 Modkit |
    | `BT_QUIRK_8BITDO_M30` | 8BitDo M30 controller |
    | `BT_QUIRK_8BITDO_M30_MODKIT` | 8BitDo M30 Modkit |
    | `BT_QUIRK_8BITDO_SATURN` | 8BitDo Saturn controller |
    | `BT_QUIRK_8BITDO_GC` | 8BitDo GameCube controller |
    | `BT_QUIRK_8BITDO_GBROS` | 8BitDo GBros. adapter |
  </Accordion>
  <Accordion title="Other third-party controllers">
    | Quirk flag | Controller |
    |------------|------------|
    | `BT_QUIRK_BLUEN64_N64` | BlueN64 N64 controller |
    | `BT_QUIRK_RF_WARRIOR` | RF Warrior controller |
    | `BT_QUIRK_OUYA` | OUYA controller |
  </Accordion>
  <Accordion title="Generic face button quirks">
    Some controllers report face buttons in a non-standard orientation. BlueRetro corrects for these automatically:

    | Quirk flag | Effect |
    |------------|--------|
    | `BT_QUIRK_FACE_BTNS_INVERT` | Inverts face button polarity |
    | `BT_QUIRK_FACE_BTNS_ROTATE_RIGHT` | Rotates face button layout clockwise |
    | `BT_QUIRK_FACE_BTNS_TRIGGER_TO_6BUTTONS` | Maps triggers to a 6-button layout |
    | `BT_QUIRK_FACE_BTNS_TRIGGER_TO_8BUTTONS` | Maps triggers to an 8-button layout |
    | `BT_QUIRK_TRIGGER_PRI_SEC_INVERT` | Swaps primary and secondary trigger assignments |
  </Accordion>
</AccordionGroup>

<Info>
  If your Bluetooth controller is not listed but presents a standard HID descriptor, connect it as a generic HID device. Use the BlueRetro web configuration interface to manually map any buttons or axes that do not align to the default layout.
</Info>
